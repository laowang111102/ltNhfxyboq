# 电影信息管理系统 python469

## 项目概述

电影信息管理系统是一款基于Python + Flask开发的Web应用，采用前后端分离的架构设计，前端使用Vue实现用户交互界面，后端则负责数据处理和业务逻辑。

## 技术栈

- 前端：Vue
- 后端：Python、Flask
- 数据采集：Scrapy
- 数据存储：MySQL
- 数据处理：Pandas

## 功能模块

### 管理员端

- 电影资讯全周期维护
- 轮播图内容调控
- 外部电影数据爬取与整合
- 深度管理电影基础信息
- 用户账号状态管控

### 用户端

- 个人资料编辑
- 电影评论与交互
- 首页推荐与动态获取
- 资讯浏览与电影信息查询
- 标准化注册登录流程

## 系统角色

- 管理员：负责系统的维护和用户管理
- 用户：进行电影信息查询、评论交互等操作

## 运行环境

- Python 3.*
- Flask
- Vue
- Scrapy
- MySQL

## 部署步骤

1. 安装Python环境
2. 部署MySQL数据库
3. 安装并配置Flask后端服务
4. 构建Vue前端项目
5. 部署Scrapy爬虫服务
6. 部署应用

## 目录结构

```
/backend - 后端代码
    /app - Flask应用
    /models - 数据模型
    /views - 视图逻辑
    /templates - 模板
/frontend - 前端代码
    /src - Vue源码
    /public - 静态文件
```

## 核心亮点

- 前后端分离，便于维护与扩展
- 利用Scrapy高效爬取外部电影数据
- 使用Pandas进行数据清洗和规整
- 基于MySQL的数据存储，稳定可靠
